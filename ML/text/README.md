# Задание

Обучите NLP модель, измерьте метрики.

1. Соберите датасет из текстов.

2. (если нет разметки) Разметьте данные.

3. Реализуйте текстовую модель.

4. Обучите модель на размеченном датасете.

5. Измерьте качество модели.

Для реализации рекомендуются эти python библиотеки:
  - pytorch
  - tensorflow
  - youtokentome для byte-pair encoding
